# Project Overview

# Setup Instructions

## Step 1:
Create a virtual environment. Install all the dependencies from requirements.txt

## Step 2:
Create a kernel of the environment, always choose this kernel while executing your python code.

## Step 3:
Open a terminal and navigate to your project folder. 
Open ../frontend in the terminal, and run
> npm start

## Step 4:
Open another terminal and navigate to your project folder.
Open the virtual environment using
> venv/Scripts/activate

Navigate to ../backend and run
> python app.py
